# Vahşi Batı: Top-Down Shooter
Bu proje, C++ ve SFML kütüphanesi kullanılarak geliştirilmiş 2 boyutlu bir hayatta kalma/nişancı oyunudur.

## Özellikler
* Dalga (Wave) tabanlı düşman sistemi
* Rastgele düşen güçlendirmeler (Can, Hız, Ateş Hızı, Bomba, Skor)
* Çarpışma kontrolleri ve animasyonlu kullanıcı arayüzü (UI)

## Kontroller
* **W, A, S, D:** Karakter Hareketi
* **Fare Sol Tık:** Ateş Etme

## Kurulum ve Çalıştırma
### Yöntem 1:
Oyunu derlemekle uğraşmadan doğrudan kurup oynamak isterseniz:
1. GitHub sayfasının sağ tarafındaki **Releases** bölümüne gidin.
2. En güncel sürümün (şuanlık v1.0.0) altındaki `Assets` kısmından **`TopDownShooter_Setup.exe`** dosyasını indirin.
3. Dosyayı çalıştırıp kurulumu tamamladıktan sonra oynamaya başlayabilirsiniz.

### Yöntem 2:
Projeyi derlemek için sisteminizde SFML ve MinGW/GCC kurulu olmalıdır. 
Terminalde proje dizinine giderek `make` komutunu çalıştırmanız yeterlidir.
